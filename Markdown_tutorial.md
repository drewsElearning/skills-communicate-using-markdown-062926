---

## 🎯 What is Markdown?

Markdown is a simple way to format text using plain characters. It allows you to create:

- Headings  
- Lists  
- Links  
- Images  
- Code blocks  
- Tables  

You write Markdown using plain text, and GitHub automatically renders it into formatted content.
